
🌐 Student Portfolio Website

A modern, responsive, and interactive Student Portfolio Website developed using HTML5, CSS3, and JavaScript. This project showcases my academic background, technical skills, projects, certifications, achievements, and contact information in a clean and professional user interface.

📖 Overview

The Student Portfolio Website serves as my digital portfolio and personal branding platform. It highlights my journey as a web development student, demonstrates my front-end development skills, and provides an easy way for recruiters, educators, and collaborators to learn more about me and my work.

✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Fixed navigation bar with smooth scrolling
- Attractive Home section with profile image
- About Me section
- Education timeline
- Technical Skills showcase
- Projects section
- Professional Certifications section
- Achievements section
- Contact Information with social media icons
- Animated typing effect
- Scroll-to-top button
- Sticky header with shadow effect
- Modern card-based UI with hover animations

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts (Poppins)
